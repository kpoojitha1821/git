# Day 1 Notes

## Arrays
- Array (List) stores multiple values.
- Elements are accessed using index.
- Index starts from 0.
- Supports slicing.

Example:
arr = [10, 20, 30, 40]

## Strings
- String is a sequence of characters.
- Strings are immutable.
- Supports indexing and slicing.

Example:
s = "Python"

## Two Pointer Pattern
- Uses two pointers (indexes).
- One starts from the left.
- One starts from the right.
- Used to solve array and string problems efficiently.

## Time Complexity
- append() → O(1)
- insert() → O(n)
